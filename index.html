DaveShade

A fast cross-api graphics layer.

DaveShade is made to make development of graphical web applications faster through easy access to any underlying graphics library the browser may have.

A demo can be seen in the backdrop of this page! The hello world is here.

DaveShade by default comes with a WebGL module and a base module that can be extended.

To get started grab the latest version, and run DaveShade.createInstance(canvas)
